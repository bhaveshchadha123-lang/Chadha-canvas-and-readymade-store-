   └─ Time: 1 minute

STEP 3: Push Code to GitHub
   └─ cd C:\Users\DELL\Desktop\chadha-canvas
   └─ git init
   └─ git add .
   └─ git commit -m "Chadha Canvas Website"
   └─ git remote add origin https://github.com/YOUR/chadha-canvas.git
   └─ git branch -M main
   └─ git push -u origin main
   └─ Time: 2 minutes

STEP 4: Deploy on Netlify
   └─ Go: https://app.netlify.com
   └─ Sign up with GitHub
   └─ Import chadha-canvas repo
   └─ Click Deploy
   └─ Time: 3 minutes

STEP 5: Deploy on Render
   └─ Go: https://render.com

   └─ Set Start: python backend/app.py
   └─ Click Deploy
   └─ Time: 3 minutes
