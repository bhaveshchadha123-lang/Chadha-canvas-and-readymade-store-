3. Configure Git locally:
```powershell
git config --global user.email "your.email@gmail.com"
git config --global user.name "Your Name"
```

### **Step 2: Push Code (2 minutes)**
```powershell
cd C:\Users\DELL\Desktop\chadha-canvas
git init
git add .
git commit -m "Chadha Canvas Website - Complete Build"
git remote add origin https://github.com/YOUR_USERNAME/chadha-canvas.git
git branch -M main
git push -u origin main
```

### **Step 3: Deploy Frontend (3 minutes)**
1. Go to https://app.netlify.com
2. Sign up with GitHub
3. Import chadha-canvas repository
4. Deploy!
